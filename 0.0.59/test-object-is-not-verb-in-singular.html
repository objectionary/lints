Test object names must start with a verb in singular form.

Incorrect:

# Foo.
[] > foo
  # This is test.
  [] +> this-is-test
    42 > foo

  # This is test.
  [] +> testing
    42 > foo

  # This is test.
  [] +> itIsTrue
    42 > foo

Correct:

# Foo
[] > foo
  # This is test.
  [] +> prints-data
    42 > foo

  # This is test.
  [] +> works-as-expected
    42 > foo

  # This is test.
  [] +> parses-dom
    42 > foo
