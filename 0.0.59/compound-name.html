Attribute Names Should Not Be Compound

All objects that are not formations must have single-word names. Compound names indicate that the program context is too large and needs to be decomposed.

Incorrect:

# CSV.
[csv-file-name] > csv-object
  parse > read-records
    read csv-file-name

Correct:

# CSV.
[file] > csv-object
  parse > records
    read file
