Schema Is Absent

XMIR document must have xsi:noNamespaceSchemaLocation inside it.

Incorrect:

<program>
  <objects/>
</program>

Correct:

<program xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="https://www.eolang.org/xsd/XMIR-anything.xsd">
  <objects/>
</program>
