Atom and Base

In XMIR, atom object can not have @base attribute.

Incorrect:

<object>
  <o name="obj" base="bar">
    <o name="λ"/>
  </o>
</object>

Correct:

<object>
  <o name="obj">
    <o name="λ"/>
  </o>
</object>

or:

<object>
  <o name="obj" base="bar">
    <o name="foo"/>
  </o>
</object>
