Unused Void Attr

Declared void attribute should be used inside the formation.

Incorrect:

# Foo.
[x] > foo
  42 > bar

Correct:

# Foo.
[x] > foo
  x > bar

Unused void attributes are allowed only if there are atoms in the program:

# Foo.
[x] > foo
  [] > bar ?

Or formation itself is the atom:

# Foo.
[x] > foo ?
