Atom Without rt Meta

Defined atom must have +rt meta.

Incorrect:

+architect jeff@foo.com
+home https://github.com/objectionary/eo
+package org.eolang
+version 0.0.0

[] > bytes
  [x] > eq /bool

  [y] > not /bool

Correct:

+architect jeff@foo.com
+home https://github.com/objectionary/eo
+package org.eolang
+version 0.0.0
+rt jvm org.eolang:eo-runtime:0.43.2

[] > bytes
  [x] > eq /bool

  [y] > not /bool
