Anonymous Formation

Anonymous formation should not access object, undefined inside the formation.

Incorrect:

# App.
[] > app
  "Hello" > t
  malloc.of
    64
    [m]
      QQ.io.stdout > @
        t

Correct:

# App.
[] > app
  "Hello" > t
  malloc.of
    64
    [m]
      "boom" > x
      QQ.io.stdout > @
        x
