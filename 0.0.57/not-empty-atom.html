Not empty atom

In XMIR, an atom (an object with an inner object where @name equals λ) must not have an inner object with @base not equal to ∅.

Incorrect:

<object>
  <o line="1" name="number">
    <o base="f" name="λ"/>
  </o>
</object>

Correct:

<object>
  <o name="number">
    <o base="∅" name="λ"/>
  </o>
</object>
