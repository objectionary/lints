The @ attribute should always go first.

Incorrect:

# Foo.
[] > foo
  x > bar
  y > @
  z > hey

Correct:

# Foo.
[] > foo
  y > @
  x > bar
  z > hey

The position of @ after void attributes is also allowed:

# Foo.
[] > foo
  [i] >>
    boom > @
      *
        "f"
