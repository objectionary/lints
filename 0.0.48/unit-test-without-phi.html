Unit Test Without Phi

Unit test must have @ attribute.

Incorrect:

+tests

# Test suite for `foo`.
[] > foo-tests
  # Unit test.
  [] > works-dummy
    true > i

Correct:

+tests

# Test suite for `foo`.
[] > foo-tests
  # Unit test.
  [] > works-dummy
    true > @
